#### Paleta customizada {#custom-palette}
Personalize la paleta de colores.

Puede obtener el código de color hexadecimal de la siguiente [página](https://htmlcolorcodes.com/).

Debe especificar el color de cada uno de los siguientes niveles, la `p` indica positivo, `n` indica negativo, del 1-3 cuanto más alto más votos hay.
* p1, Establece el color positivo para pocos votos.
* p2, Establece el color positivo para la mitad de votos.
* p3, Establece el color positivo para muchos votos.
* n1, Establece el color negativo para pocos votos.
* n2, Establece el color negativo para la mitad de votos.
* n3, Establece el color negativo para muchos votos.

## Lockdown {#lockdown}

El lockdown le permite bloquear distintos canales a la vez para que los usuarios no puedan enviar mensajes en esos canales, como el `lock` pero en varios canales.

* Usa `/lockdown add` para añadir un canal a la lista de canales bloqueados.
* Usa `/lockdown remove` para eliminar un canal de la lista de canales bloqueados.
* Usa `/lockdown list` para ver la lista de canales bloqueados.
* Usa `/lockdown toggle` para activar/desactivar el lockdown.

## Bloqueador de enlaces {#block-links}

El bloqueador de enlaces le permite bloquear un enlace específico para evitar que sea enviado, esto es útil si quiere bloquear un enlaces específico que no está en alguno de los [filtros de enlaces](config/filters.mdx#link-filters).

* Usa `/filter links add` para añadir un enlace a la lista de enlaces bloqueados.
* Usa `/filter links remove` para eliminar un enlace de la lista de enlaces bloqueados.
* Usa `/filter links list` para ver la lista de enlaces bloqueados.
