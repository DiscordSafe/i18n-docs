## Acciones en masa {#mass-actions}
Puedes aceptar, denegar y borrar sugerencias en masa si lo deseas, estos comandos no admiten comentarios.

* Aceptar: `/suggestions massaction action:Approve suggestionids:1,2,3`.
* Denegar: `/suggestions massaction action:Deny suggestionids:1,2,3`.
* Borrar: `/suggestions massaction action:Delete suggestionids:1,2,3`.
