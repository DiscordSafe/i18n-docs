## Establecer el canal de sugerencias {#set-channel}
Establezca el canal donde se enviarán las sugerencias de los usuarios.
Establézcalo con `d.suggestions setchannel #Canal/CanalID`.

## Activar el emoji de indeciso {#set-question-emoji}
Puede indicar que se añada el emoji de indeciso `❓` a las reacciones de las sugerencias, para aquellos usuarios que no estén seguros.

![Ejemplo](https://i.imgur.com/ARFrHal.png)

Puedes configurarlo con `d.suggestions questionEmoji enable / disable`.

## Auto-aceptar sugerencias {#autoaccept-suggestions}
Puede establecer una cantidad de votos positivos donde las sugerencias se aceptan automáticamente.

Configúrelo con `d.suggestions autoaccept Número(1-1000) / disable`.

## Aceptar, denegar y borrar sugerencias {#approve-deny-delete}
:::info
La ID de la sugerencia la puede encontrar al final del mensaje de la sugerencia, estos son números únicos para cada sugerencia.

![Ejemplo](https://i.imgur.com/y0wOy7i.png)
:::

* Aceptar: `d.suggestions approve IDSugerencia <Comentario>`.

## Acciones en masa {#mass-actions}
Puedes aceptar, denegar y borrar sugerencias en masa si lo deseas, estos comandos no admiten comentarios.

* Aceptar: `d.suggestions massapprove IDSugerencia IDSugerencia...`.
* Denegar: `d.suggestions massdeny IDSugerencia IDSugerencia...`.
* Borrar: `d.suggestions massdelete IDSugerencia IDSugerencia...`.

## Paleta de colores {#color-palette}
Puede customizar los colores de las sugerencias, los colores cambian de nivel dependiendo del número de votos.

Puede elegir entre una de las siguientes paletas de colores:

![Paletas de colores](https://i.imgur.com/e0wPvwV.png)

Puede seleccionar la peleta de colores que desea usar con `d.suggestions setpalette <Nombre paleta>`.
