## Establecer el canal de sugerencias {#set-channel}

Establézcalo con `d.suggestions setchannel #Canal/CanalID`.

## Acciones en masa {#mass-actions}
Puedes aceptar, denegar y borrar sugerencias en masa si lo deseas, estos comandos no admiten comentarios.

* Aceptar: `d.suggestions massapprove IDSugerencia IDSugerencia...`.
* Denegar: `d.suggestions massdeny IDSugerencia IDSugerencia...`.
* Borrar: `d.suggestions massdelete IDSugerencia IDSugerencia...`.

## Paleta de colores {#color-palette}
Puede customizar los colores de las sugerencias, los colores cambian de nivel dependiendo del número de votos.

Puede elegir entre una de las siguientes paletas de colores:

![Paletas de colores](https://i.imgur.com/e0wPvwV.png)

Puede seleccionar la peleta de colores que desea usar con `d.suggestions setpalette <Nombre paleta>`.

:::info
También puede customizar cada valor y crear su propia paleta de colores, eso requiere **D-Safe Premium**, estará a continuación.
:::

### Paleta customizada {#custom-palette}
Customize la paleta de colores con `d.suggestions setpalette p1 #ColorHexadecimal`.

Puede obtener el código de color hexadecimal de la siguiente [página](https://htmlcolorcodes.com/).

Debe especificar el color de cada uno de los siguientes niveles, la `p` indica positivo, `n` indica negativo, del 1-3 cuanto más alto más votos hay.
* p1, Establece el color positivo para pocos votos.
* p2, Establece el color positivo para la mitad de votos.
* p3, Establece el color positivo para muchos votos.
* n1, Establece el color negativo para pocos votos.
* n2, Establece el color negativo para la mitad de votos.
* n3, Establece el color negativo para muchos votos.
