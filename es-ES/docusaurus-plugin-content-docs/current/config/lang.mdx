El bot usará el idioma que tenga establecido en los ajustes de su servidor *(si tiene comunidad activado)*, puede verlo en Ajustes del servidor > Comunidad > Vista general.

Le recomendamos que indique el idioma del bot desde esta manera.

![img](https://i.imgur.com/otn7DBg.png)

Si quiere indicar manualmente el idioma del bot por alguna razón puede usar el comando `/setlang`.

Si no hay un idioma definido de ninguna manera, el idioma por defecto es **Español.**

## Idiomas disponibles {#available-locales}
* ``es-ES`` - Español (España).
* ``en-US`` - Inglés (Estados Unidos).
