El idioma por defecto del bot es **Español.**

## Idiomas disponibles {#avalable-locales}
* ``es-ES`` - Español (España).
* ``en-US`` - Inglés (Estados Unidos).

Para cambiar el idioma use `d.setlang [es-ES / en-US]`.
