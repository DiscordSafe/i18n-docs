Configure el aspecto de moderación del bot.


## Roles de moderadores {#mod-roles}

El bot por defecto solo permitirá usar los [comandos de moderación](basics/commands.md#mod) a los usuarios con permiso de **Administrador**, **Banear miembros** o **Gestionar mensajes**, si quieres permitir a un rol/roles usarlos sin tener que configurar los permisos puedes configurar los roles de moderadores.


* Añadir un rol: `/modroles add`.
* Quitar un rol: `/modroles remove`.
* Mostrar la lista de roles de moderadores: `/modroles list`.

## Notificar al sancionado al sancionarle {#notifyUsers}

Por defecto el bot no notificará a los usuarios al sancionarles.

* Para configurarlo globalmente usa `/setup` y elige "Moderación".
* Para hacerlo en una sanción seleccionalo a las opciones del comando.

## Borrar mensajes después de usar comandos de moderación {#deleteCmds}

Por defecto el bot no borrará los mensajes después de usar los comandos de moderación.

* Para configurarlo globalmente usa `/setup` y elige "Moderación".
* Para hacerlo en una sanción seleccionalo a las opciones del comando.
