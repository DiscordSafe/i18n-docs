Los filtros le permiten borrar todo el contenido que no quiera que se mande en su servidor.

Use `/filter setup` para seleccionar los filtros que desea activar.

![img](https://i.imgur.com/7BDt5Uq.gif)

## Filtros de mensajes

* **Flood**, borra mensajes enviados rápidamente.
* **Palabras repetidas**, borra mensajes con muchas palabras repetidas.
* **Mayúsculas**, borra mensajes con exceso de mayúsculas.
* <discord-custom-emoji name="premiumsparkles"></discord-custom-emoji> **Imágenes**, borra imágenes con contenido explícito fuera de canales NSFW.

### Cooldowns

Los filtros de cooldown le permiten limitar el envío de un contenido a una cantidad por segundos.

* **Cooldown de enlaces**, limita el envío de enlaces a una cantidad por segundos.
* **Coldown de invitaciones**, limita el envío de invitaciones a una cantidad por segundos.
* **Cooldown de imágenes**, limita el envío de imágenes a una cantidad por segundos.

## Filtros de enlaces

Los filtros de enlaces le permiten borrar los tipos de enlaces no permitidos.

* **Invitaciones**, borra invitaciones a otros servidores de Discord.
* **IPloggers**, borra enlaces maliciosos que recopilan direcciones IP de los usuarios.
* **Estafas**, borra enlaces fraudulentos con motivos económicos.
* **Phishing**, borra enlaces que suplantan a Discord y otras empresas para recopilar información y/o robar cuentas.

## Ignorar canales o roles {#ignore}

¿Quieres que un filtro ignore canales o roles? No te preocupes, puedes hacerlo fácilmente.

__Para dejar de ignorar un canal o rol, vuelve a usar el mismo comando.__

### Ignorar roles {#ignore-roles}
Usa `/filter ignore role` para ignorar roles.

  <discord-message profile="bot">
    <discord-command slot="reply" profile="user" command="/filter ignore role">

</DiscordMessages>
