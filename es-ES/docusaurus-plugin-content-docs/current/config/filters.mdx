Los filtros permiten proteger tu servidor para **evitar que se envíe contenido malicioso** y que tus canales sean un lugar limpio.

### Activar {#activar}
Para activarlos siga la guía dando click abajo, al filtro que quiera activar.

### Desactivar {#desactivar}
Para desactivar un filtro use: ``d.filters disable <filtro>``

  Analiza los mensajes y si detecta que es spam lo borrará.
  

  **Para activarlo usa:** ``d.filter enable anti-spam``
  

  ## Niveles:

  Puedes indicar el nivel de **anti-spam** que quiera, por defecto o si no lo indica será el ``1``.
  
  Establece el nivel con: ``d.filter enable anti-spam level <Nivel>``

  | 📏 Nivel | 📣 Descripción |
  | :---: | :--- |
  | <img className="emoji-table" src="https://cdn.discordapp.com/emojis/755768668360540170.png?v=1"></img> **Nivel 1** | Prohíbe un mensaje con muchas mayúsculas o spam. | 
  | <img className="emoji-table" src="https://cdn.discordapp.com/emojis/755768668180316202.png?v=1"></img> **Nivel 2** | Prohíbe los mensajes duplicados, flood de mensajes, mayúsculas y spam. | 
  | <img className="emoji-table" src="https://cdn.discordapp.com/emojis/755768668318728202.png?v=1"></img> **Nivel 3** | Prohíbe todos los mensajes duplicados, flood de mensajes, mayúsculas, copypastas y spam. |
