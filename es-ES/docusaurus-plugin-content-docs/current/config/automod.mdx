La automoderación le permite moderar automáticamente cuando se comentan infracciones a los [filtros](./filters).

El objetivo de D-Safe es ofrecer una moderación automática avanzada pero siendo sencilla y accesible para todos.

La configuración de la moderación automática es importante que sea configurada dependiendo de las necesidades y cantidad de miembros del servidor.

## ¿Por qué usar moderación automática? {#why}
Si tiene un servidor bien definido, es probable que ya tenga un equipo de moderadores, quizá se pregunte, ¿por qué debo usar moderación automática si ya tengo moderadores?

La moderación automática no reemplaza la moderación manual, es algo que la mejora y enriquece. Su equipo de moderación puede seguir moderando basándose en el contexto y la moderación automática le facilitará el proceso encargándose de problemas comunes más rápido de lo que podría hacerse manualmente.

## Términos {#terms}

* **Castigo:** configuración de una __sanción__ cuando se comentan __X infracciones__ de __Y filtros__ en __Z tiempo__.
* **Infracción:** actividad realizada por un usuario que infringe algún filtro.
* **Sanción:** acción que se le aplica al usuario.

## Configuración recomendada {#recommended-config}

¿No quieres pasar mucho tiempo configurando y quieres algo básico? ¡Usa la configuración recomendada!

Usa `/setup` y selecciona "Auto Moderación" para elegir los castigos que quieres en tu servidor, por defecto seleccionados los recomendados.

## Configuración manual {#manual-config}

Si quieres puedes añadir manualmente un castigo completamente customizable usando `/punishment add`.

### Lista de castigos {#punishment-list}

Usa `/punishment list` para ver la lista de castigos.

### Quitar un castigo {#punishment-remove}

Usa `/punishment remove` para quitar un castigo, debes usar el número del castigo que puedes ver en la lista.
