Puede acceder a todos los comandos disponibles en el bot usando los <abbr title="Comandos especiales que empiezan por /">comandos de barra diagonal</abbr> sin necesidad de usar o configurar un prefijo.

Pero si desea cambiar el prefijo de los comandos normales vía mensaje puede usar `/setprefix` o `d.setprefix [nuevo prefix]`.

<DiscordMessages>
  <discord-message profile="bot">
    <discord-command slot="reply" profile="user" command="/setprefix">
    </discord-command>
    <discord-custom-emoji name="checknew"></discord-custom-emoji> | El prefix ha sido establecido a <code>!!</code>.
  </discord-message>
</DiscordMessages>

<br/>

:::tip
**¿No te acuerdas del prefijo?**
Usa ``@D-Safe prefix`` para ver el prefijo de tu servidor.
:::
