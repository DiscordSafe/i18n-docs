La nueva versión de D-Safe introduce el comando `setup` que le permite configurar al bot de manera sencilla usando reacciones o botones como se muestra a continuación.

Este nuevo comando le permite configurar por el momento:
* Verificación.
* Filtros.
* Moderación.
