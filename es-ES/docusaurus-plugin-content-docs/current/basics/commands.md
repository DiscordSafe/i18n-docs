* * * 
Lista de comandos de D-Safe.

Recuerda que no debes poner los siguientes símbolos al usar el comando.
* ` / ` Divide y separa las opciones de ese comando.
* `[]` Indica que el argumento es requerido.
* `<>` Indica que el argumento es opcional.

## Configuración {#config}

| Nombre       | Descripción                                                             | Uso                                                                                                        | Ejemplo                              | Permisos                    |
| ------------ | ----------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------- | ------------------------------------ | --------------------------- |
| blockword    | Bloquea el uso de palabras en el servidor.                              | `add [Palabra/Pack] / remove [Palabra/Pack] / list / reset`                                                | `d.blockword add pack 0`             | `MANAGE_MESSAGES`           |
| clearwarn    | Elimina una advertencia del servidor.                                   | `<Código> / all`                                                                                           | `d.clearwarn 43das78`                | `BAN_MEMBERS`               |
| filter       | Configura los filtros.                                                  | `help / activate <filtro> / deactivate <filtro>`                                                           | `d.filter activate anti-spam`        | `MANAGE_ROLES MANAGE_GUILD` |
| logs         | Establece el canal de logs del servidor.                                | `<#Canal> / <IDCanal> / none`                                                                              | `d.logs #Logs`                       | `MANAGE_GUILD`              |
| notification | Configura los avisos de entrada de usuarios maliciosos.                 | `on / off`                                                                                                 | `d.notification on`                  | `ADMINISTRATOR`             |
| setlang      | Establece el idioma del bot en el servidor.                             | `es-ES / en-US`                                                                                            | `d.setlang es-ES`                    | `MANAGE_GUILD`              |
| setprefix    | Establece el prefijo en el servidor.                                    | `[prefijo]`                                                                                                | `d.setprefix !!`                     | `MANAGE_GUILD`              |
| settings     | Muestra los ajustes del servidor e información sobre cómo configurarlo. | `prefix / announcements / blockedwords / whitelist / logs / lang / verification / filters / notifications` | `d.settings verificación`            | `BAN_MEMBERS`               |
| whitelist    | Configura la Whitelist de su servidor.                                  | `add <ID> / remove <ID> / list / reset`                                                                    | `d.whitelist add 461171501715161108` | `BAN_MEMBERS`               |
| report | Reporta a un usuario para ser añadido al forceban. |  | `d.report .sknahTT#1785 318327338226155541 Hacer SPAM al md https://imgur.com/linkfalso` |        |

## Moderación {#mod}

| Nombre   | Descripción                                              | Uso                                                      | Ejemplo                                              | Permisos          |
| -------- | -------------------------------------------------------- | -------------------------------------------------------- | ---------------------------------------------------- | ----------------- |
| ban      | Banea a un usuario del servidor.                         | `<@Usuario / ID> <Razón>`                                | `d.ban @Vicente#0001 Romper la regla 4.`             | `BAN_MEMBERS`     |
| clear    | Borra una cantidad de mensajes.                          | `<Cantidad(2-100)> / <@Usuario> <Cantidad(2-100)> / all` | `d.clear @Seyron#5532 9`                             | `MANAGE_MESSAGES` |
| forceban | Banea a los usuarios peligrosos de mi base de datos.     |                                                          |                                                      | `MANAGE_GUILD`    |
| hackban  | Banea a uno o varios usuarios mediante su ID.            | `[ID] <ID> <ID>...`                                      | `d.hackban 643945264868098049`                       | `BAN_MEMBERS`     |
| kick     | Expulsa a un miembro del servidor.                       | `<@Miembro> <Razón>`                                     | `d.kick Discord#0000 Romper la regla 8.`             | `KICK_MEMBERS`    |
| mute     | Silencia a un miembro                                    | `<@Miembro> [Tiempo] [s/m/h] <Razón>`                    | `d.mute @Clyde#0001 40 m Romper la regla 17.`        | `MANAGE_ROLES`    |
| unban    | Desbanea a un usuario del servidor.                      | `<@Usuario / ID>`                                        | `d.unban 287574458963722240`                         | `BAN_MEMBERS`     |
| unmute   | Desilencia a un miembro silenciado.                      | `<@Miembro / ID>`                                        | `d.unmute 182866681520062465`                        | `MANAGE_ROLES`    |
| warn     | Advierte a un usuario de su mal comportamiento.          | `<@Usuario> <Razón>`                                     | `d.warn @[Dracule Mihawk™ ]#4449 Romper la regla 3.` | `KICK_MEMBERS`    |
| warnings | Obtiene la lista de advertencias del servidor o miembro. | `<@Usuario>`                                             | `d.warnings @Vicente#0001`                           | `MANAGE_MESSAGES` |
